
# Heart Disease Prediction Streamlit App

Bu proje BIM 322 Makine Öğrenmesi ve Uygulamaları dersi kapsamında geliştirilmiştir.

## Proje Konusu

Heart Disease Cleveland veri seti kullanılarak kalp hastalığı riski tahmini yapılmıştır.

## Kullanılan Final Model

Support Vector Machine

## Test Sonuçları

- Accuracy: 0.8913
- Precision: 0.8333
- Recall: 0.9524

## Çalıştırma

Aşağıdaki komutlar kullanılarak uygulama çalıştırılabilir:

pip install -r requirements.txt

streamlit run app.py

## Not

Bu uygulama yalnızca eğitim amaçlıdır. Tıbbi teşhis veya tedavi amacıyla kullanılmamalıdır.
